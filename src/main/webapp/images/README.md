# Logo Image

Please save the Ocean View Resort logo image as **`logo.png`** in this directory.

The logo should be the colorful beach scene with:

- Palm trees
- Sunset/sun illustration
- Ocean waves
- "Ocean View" text in blue
- "Resort" text in red
- "Galle" text in blue

The image has been configured to display at:

- **50px height** in the navbar
- **80px height** in sidebars

Once you save the logo.png file here, it will automatically appear throughout the application.
